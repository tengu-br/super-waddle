# super-waddle

Código base para hospedar múltiplos sites wordpress de maneira rápida e eficaz utilizando docker.

# Resolvido

* Compartilhamento de banco de dados
* Proxy reverso
* MariaDB
* Repositório auxiliar com temas e plugins
